# Url Mock

A simple library for mocking connections obtained through `java.net.URL`

## Installation
```xml
<dependency>
  <groupId>io.bestquality</groupId>
  <artifactId>urlmock</artifactId>
  <version>0.0.18</version>
  <scope>test</scope>
</dependency>
```
